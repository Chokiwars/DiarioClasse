# 📘 Diário de Classe

Sistema para gerenciamento de atividades escolares.

## 🚀 Como executar o projeto

### Pré-requisitos
- Node.js (versão 16 ou superior)
- Git

### Instalação

```bash
# Clone o repositório
git clone https://github.com/sabrinasavini/DiarioClasse.git

# Acesse a pasta
cd DiarioClasse

# Instale as dependências
npm install

# Execute o projeto
npm start
```
